# Prompt maestro — pegar al inicio de cada chat

Copia y pega el bloque siguiente **al abrir un chat nuevo** con la IA. Así das contexto en pocas líneas y ahorras tokens (no tienes que repetir quién eres, qué es AVENTA ni qué stack usas).

---

## Bloque para copiar

```
Contexto: Soy Jafet, llevo AVENTA (comunidad de ofertas en México). No programo; uso IA y producción para lo técnico. Stack: Next.js, Supabase, Vercel, Upstash. Estado y checklist: docs/GUIA_AVENTA.md. Día a día: docs/COMO_LLEVAR_AVENTA.md. Pendientes: docs/PENDIENTES.md o la sección en GUIA. Si necesitas más detalle de código o DB, pregunta o revisa docs/ (SUPABASE_CONTEXTO.md, SISTEMAS_AVENTA.md).
```

---

## Uso

- **Chat nuevo:** Pega el bloque arriba y debajo escribe lo que quieres hacer (ej. “arregla el bug de X”, “actualiza el copy de Y”, “revisa PENDIENTES.md”).
- **Misma sesión:** No hace falta repetir; la IA ya tiene el contexto del chat.
- **Cambios de proyecto:** Si cambias stack o docs clave, actualiza este bloque aquí y en COMO_LLEVAR_AVENTA.md si lo mencionas.
