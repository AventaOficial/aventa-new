# Prompts para generar logo AVENTA

Usa estos prompts en generadores de logos con IA (ver recomendaciones al final).

---

## Prompt 1 — Letra "A" minimalista

```
Logo minimalista para marca "AVENTA". La letra "A" como elemento central, tipografía geométrica y limpia. Estilo premium, refinado, calidad de producto de alta gama. Colores: violeta/púrpura suave como acento, fondo neutro. Sin elementos decorativos extra. Vector, escalable, apto para app y web.
```

---

## Prompt 2 — Cazadores de ofertas

```
Logo para app de comunidad cazadora de ofertas. Concepto: ahorro, descubrimiento, comunidad. Estilo minimalista, premium, calidad de producto refinado. Sin iconos genéricos de cupones. Sugerencia sutil: flecha hacia abajo (precio), lupa (descubrir) o forma abstracta que evoque "encontrar". Tipografía limpia, colores violeta y neutros. Vector, escalable.
```

---

## Dónde usarlos

- **Looka** (looka.com) — Generador de logos con IA, exporta en varios formatos.
- **Canva AI** (canva.com) — Herramienta de diseño con generador de logos.
- **Brandmark** (brandmark.io) — Especializado en logos con IA.
- **Hatchful by Shopify** (hatchful.shopify.com) — Gratuito, buenos resultados minimalistas.

Recomendación: probar primero en **Looka** o **Brandmark** para resultados más premium y minimalistas.
